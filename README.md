# Klein Shell DOS 2.0 Beta 2

A retro terminal educational game by **Rodrigo Klein Mariano Canto / Klein Dream**, created with assistance from **Lolita / ChatGPT**.

This project is inspired by the golden age of command-line computing, but it is an original Klein Dream creation. It is not affiliated with, endorsed by, or presented as a product of any other company.

## What is new in Beta 2

- 20 playable story missions
- New hidden areas: `C:\DEEP`, `C:\TRACE` and `C:\VAULT`
- New tools: `DEFRAG`, `VERIFY` and `SHIELD`
- Expanded ending with a stronger Klein Dream message
- More ethical, educational and human-centered narrative
- Cleaner project identity: Klein Dream Terminal
- Themes: green, amber and blue
- Borderless 1366x768 Electron window
- Compatible project setup for Windows 7 + Node 12

## Running from source

```bash
npm install
npm start
```

## Building portable EXE

```bash
npm run dist
```

Recommended environment for the author's machine:

- Windows 7 x64 SP1
- Node.js 12.22.12
- Electron 6.1.12
- electron-builder 20.44.4
